# Roadmap

ZeroPing is actively evolving with a focus on stability first, then ecosystem growth.

## v1.1 ✅

- Documentation search indexing with fuzzy matching, AJAX endpoint, and local recent searches
- Starter templates (`php zero new`) for blog, API, MVC, and empty project patterns
- Expanded CLI ergonomics (`new`, `search:index` commands)
- Enhanced validation (`array`, `file`, `image`, `mimes`, `size`, `in`, `not_in`, `regex` rules, FluentValidator, FormRequest)
- Performance optimizations (view caching, route caching, config caching, lazy service loading)

## v1.2

- Package ecosystem and plugin hooks
- Deployment blueprints
- Improved observability and debug tooling

## Long-term direction

- First-party cloud workflows
- Team-oriented DX features
- Advanced code generation pipelines
