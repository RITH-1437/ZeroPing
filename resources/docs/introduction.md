# ZeroPing

ZeroPing is a zero-dependency, enterprise-grade PHP framework built for
speed, clarity, and full control. Every component — from the dependency
injection container to the ORM — is hand-written with no external packages
required at runtime.

## Highlights

- **Zero dependencies** for the runtime core.
- **Dependency Injection** container with automatic resolution.
- **Routing** with dynamic `{param}` segments.
- **ORM** with query builder, relationships, and opt-in soft deletes.
- **Testing** harness compatible with PHPUnit.
