## GitHub Discussions

For questions, ideas, and general feedback, use [GitHub Discussions](https://github.com/rith-1437/ZeroPing/discussions):

## Issues

For bug reports and concrete feature requests, use [GitHub Issues](https://github.com/rith-1437/ZeroPing/issues):

- **Bug reports** — Use the bug report template with steps to reproduce
- **Feature requests** — Use the feature request template with a concrete proposal
- **Documentation issues** — Report missing or incorrect documentation
